Detailed Description
--------------------

This file defines a group of macros that serve as attributes indicating the type of ownership of a pointer. The macros are modeled after similar attributes defines in Integer Set Library (isl) and serve the purpose of helping a programmer understand the semantics of a DPCTL function.
