Detailed Description
--------------------

This header declares a C interface to sycl::queue member functions. Note that sycl::queue constructors are not exposed in this interface. Instead, users should use the functions in :ref:`file_dpctl-capi_include_dpctl_sycl_queue_manager.h`.
