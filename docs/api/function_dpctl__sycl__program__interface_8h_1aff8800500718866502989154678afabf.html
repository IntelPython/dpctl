DPCTLSyclKernelRef DPCTLProgram_GetKernel(DPCTLSyclProgramRef PRef, const char *KernelName)¶

Returns the SyclKernel with given name from the program, if not found then return NULL.

Return

A SyclKernel reference if the kernel exists, else NULL

Parameters

  • PRef: Opaque pointer to a sycl::program

  • KernelName: Name of kernel
