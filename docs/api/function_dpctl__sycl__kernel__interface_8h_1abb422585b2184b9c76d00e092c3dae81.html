const char *DPCTLKernel_GetFunctionName(const DPCTLSyclKernelRef KRef)¶

Returns a C string for the kernel name.

Return

If a kernel name exists then returns it as a C string, else returns a nullptr.

Parameters

  • KRef: DPCTLSyclKernelRef pointer to an OpenCL interoperability kernel.
