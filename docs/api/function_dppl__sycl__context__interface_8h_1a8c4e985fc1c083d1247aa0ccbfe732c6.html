bool DPPLContext_AreEq(const DPPLSyclContextRef CtxRef1, const DPPLSyclContextRef CtxRef2)¶

Checks if two DPPLSyclContextRef objects point to the same sycl::context.

Return

True if the underlying sycl::context are same, false otherwise.

Parameters

  • CtxRef1: First opaque pointer to the sycl context.

  • CtxRef2: Second opaque pointer to the sycl context.
