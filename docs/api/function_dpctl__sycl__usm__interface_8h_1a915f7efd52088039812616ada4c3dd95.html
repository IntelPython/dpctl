DPCTLSyclUSMRef DPCTLaligned_alloc_host(size_t alignment, size_t size, const DPCTLSyclQueueRef QRef)¶

Create USM host memory.

Return

The pointer to USM host memory with the requested alignment. On failure, returns nullptr.

Parameters

  • alignment: Allocation’s byte alignment

  • size: Number of bytes to allocate

  • QRef: Sycl queue reference to use in allocation
