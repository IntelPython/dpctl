void DPCTLfree_with_queue(DPCTLSyclUSMRef MRef, const DPCTLSyclQueueRef QRef)¶

Free USM memory.

USM pointer must have been allocated using the same context as the one used to construct the queue.

Parameters

  • MRef: USM pointer to free

  • QRef: Sycl queue reference to use.
