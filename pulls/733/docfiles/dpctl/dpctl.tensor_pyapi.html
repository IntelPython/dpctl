dpctl.tensor

Data Parallel Tensor Collection is a collection of tensor implementations that implement Python data API (https://data-apis.github.io/array-api/latest/) standard.

Classes

Device

Class representing Data-API concept of device.

usm_ndarray

See dpctl.memory.MemoryUSMShared for allowed keyword arguments.

Array Construction

asarray()

Converts obj to dpctl.tensor.usm_ndarray.

asnumpy()

Copies content of dpctl.tensor.usm_ndarray instance usm_ary into numpy.ndarray instance of the same shape and same data type.

astype()

Returns a copy of the array, cast to a specified type.

copy()

Creates a copy of given instance of usm_ndarray.

empty()

Creates usm_ndarray from uninitialized USM allocation.

from_dlpack()

dpctl.tensor.from_dlpack(obj) Constructs dpctl.tensor.usm_ndarray instance from a Python object obj that implements __dlpack__ protocol. The output array is always a zero-copy view of the input.

from_numpy()

Creates dpctl.tensor.usm_ndarray from instance of numpy.ndarray.

to_numpy()

Copies content of dpctl.tensor.usm_ndarray instance usm_ary into numpy.ndarray instance of the same shape and same data type.

Array Manipulation

reshape()

-> usm_ndarray Reshapes given usm_ndarray into new shape. Returns a view, if possible, a copy otherwise. Memory layout of the copy is controlled by order keyword.
